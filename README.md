# Simple E-Commerce Website

A responsive two-page e-commerce website built with HTML, CSS, and JavaScript.

## Features

- **Homepage**: Hero section with featured products
- **Products Page**: Full product catalog with search and filter
- **Shopping Cart**: Add/remove items, persistent storage
- **Responsive Design**: Works on all devices
- **Modern UI**: Clean and professional design

## Pages

1. **index.html** - Homepage with featured products
2. **products.html** - Full product catalog

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (ES6+)
- Local Storage API

## Setup

1. Download all files into a folder
2. Open `index.html` in your browser
3. No server required - works locally
